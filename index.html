궁금한 점이 있으신가요?

❓ 어떤 형식의 영상을 업로드할 수 있나요?

MP4, MOV, AVI 형식을 지원하며, 스마트폰으로 촬영한 영상이면 충분합니다. 화질은 720p 이상을 권장하며, 선수의 등번호가 보이는 각도로 촬영해주세요.

❓ AI 분석은 얼마나 정확한가요?

10X는 최신 컴퓨터 비전과 머신러닝 기술을 사용하여 프로 분석가 수준의 정확도(90% 이상)를 제공합니다. 실제 프로 클럽에서도 사용하는 기술을 유소년 선수들에게 제공합니다.

❓ GPS 트래커도 사용할 수 있나요?

네! 현재는 영상 분석을 제공하며, GPS 데이터 연동 기능을 개발 중입니다. 곧 GPS와 영상을 결합한 더욱 상세한 분석을 제공할 예정입니다.

❓ 팀 전체 분석도 가능한가요?

개인 선수 분석뿐만 아니라 팀 전체 분석도 가능합니다. 클럽이나 아카데미를 위한 별도의 엔터프라이즈 플랜도 준비 중이니 문의 주세요.

❓ 환불 정책은 어떻게 되나요?

첫 분석 결과에 만족하지 못하시면 7일 이내 100% 환불해드립니다. 고객 만족이 최우선입니다.

⚙️ Firebase & Roboflow 설정 가이드

1️⃣ Firebase 프로젝트 설정

① Firebase Console에 로그인

② 프로젝트 생성 → "10X" (또는 원하는 이름)

③ 프로젝트 설정 → 일반 탭 → "내 앱" → 웹 앱 추가 ( 아이콘)

④ firebaseConfig 코드를 복사하여 아래 위치에 붙여넣기:

// 파일: index.html, 라인 ~450
const firebaseConfig = {
  apiKey: "YOUR_API_KEY",
  authDomain: "your-project.firebaseapp.com",
  projectId: "your-project-id",
  storageBucket: "your-project.appspot.com",
  messagingSenderId: "123456789",
  appId: "1:123456789:web:abc123"
};

2️⃣ Firebase Authentication 설정

① Firebase Console → Authentication → 시작하기

② Sign-in method 탭 → Google 제공업체 사용 설정

3️⃣ Firebase Storage 설정

① Firebase Console → Storage → 시작하기

② 규칙 탭에서 다음 규칙 설정 (보안 규칙):

rules_version = '2';
service firebase.storage {
  match /b/{bucket}/o {
    match /videos/{userId}/{allPaths=**} {
      allow read, write: if request.auth != null 
                         && request.auth.uid == userId;
    }
  }
}

4️⃣ Firestore 설정

① Firebase Console → Firestore Database → 데이터베이스 만들기

② 프로덕션 모드로 시작

③ 규칙 탭에서 다음 규칙 설정:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /analyses/{userId}/{document=**} {
      allow read, write: if request.auth != null 
                         && request.auth.uid == userId;
    }
  }
}

5️⃣ Roboflow API 키 설정

① Roboflow에 로그인

② 워크스페이스 → Settings → API 키 복사

③ 아래 위치에 붙여넣기:

// 파일: index.html, 라인 ~490
const ROBOFLOW_API_KEY = "YOUR_ROBOFLOW_API_KEY";
